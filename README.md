# 🌦️ Weather API Dashboard | Power BI

## 📌 Project Overview

This project demonstrates how to integrate weather data from an API and transform it into an interactive Power BI dashboard. The dashboard provides real-time weather insights and visual analytics to help users monitor weather conditions efficiently.

The project showcases skills in API integration, data transformation, data modeling, DAX, and dashboard design using Power BI.

---

## 🚀 Key Features

* Real-time Weather Data Integration using API
* Interactive Power BI Dashboard
* Temperature Analysis
* Humidity Monitoring
* Wind Speed Tracking
* Weather Condition Insights
* Dynamic Filters and Slicers
* User-Friendly Data Visualization

---
